# AI与低空经济资讯Agent

自动收集AI和低空经济领域的每日资讯，并发送到指定邮箱。

## 功能特点
- 📰 每日自动收集AI和低空经济新闻
- 🔍 来源包括OpenAI、机器之心、中国经济网等权威媒体
- 📧 自动发送HTML格式邮件到QQ邮箱
- ⏰ 每天北京时间7点自动运行
- 🔧 支持手动触发

## 使用方法
1. 在GitHub仓库设置Secrets（邮箱和授权码）
2. 脚本会自动运行，无需额外操作
3. 每天7点收到资讯邮件

## 修改配置
编辑 `news_collector.py` 文件中的：
- `NEWS_SOURCES`: 修改新闻源
- `KEYWORDS`: 修改过滤关键词
